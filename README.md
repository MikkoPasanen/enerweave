## About

A web app which is designed to help people in Germany to tackle the oncoming energy law changes and exchange electricity. </br>
The user can select the owned hardwares (for example Heat pump from Mitsubishi and Wallbox from Tesla) and then Enerweave will recommend the most suitable HEMS (Home energy management system) app for the user that would support as many
hardwares as the user has.
